## Sky2D-Game-Engine
A 2D Java Game Engine 
Remember to add res/ to the java build path when compiling!

### Features:
- Keyboard & Mouse Input
- Game States
- Rendering using Java AWT
- Sprite Sheets & Image Loading
- Custom Fonts
- Image Rotation
- Custom Resolutions
- Screen transitions (new)
- Animation (new)
